### .vimrc
1. **Install Vundle plugin:**  
`git clone https://github.com/VundleVim/Vundle.vim.git ~/.config/nvim/bundle/Vundle.vim`  
2. **Add some plugins to .vimrc**  
3. **Open vim and run:**  
`:PluginInstall`  

### i3 config
